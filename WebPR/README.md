#web版报价表原型
##特别说明
1. 此仅仅为原型DEMO,前端输入验证没有实现(繁琐)

##界面显示
1. 初始状态，颜色，净度下拉列表均显示--NULL--
2. 输入重量，获取对应的下拉列表

##输入重量，加载颜色和净度下拉列表
1. 请求的时机,仅当最后输入的是数字时，才发出请求，因为'.'虽然是合法的实数字符，但是'12.'不能算作完整的实数，没有必要去进行一次验证
2. 为了后台程序员的可读性和可维护性，颜色，净度下拉列表传递的值为颜色、净度的名字(如'E'、'VS1'等等)
